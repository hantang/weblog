typecho搭建

参考文档：

  • https://docs.typecho.org/install

  • 前置准备：nginx, mariadb(或myql), php

  • 数据库

sudo mariadb
# 新建数据库和用户
> CREATE DATABASE typecho DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
> CREATE USER 'typechouser'@'localhost' IDENTIFIED BY 'password';
> GRANT ALL ON typecho.* TO 'typechouser'@'localhost';
> FLUSH PRIVILEGES;
> exit;
  • 环境
# 下载
curl -LO https://github.com/typecho/typecho/releases/latest/download/typecho.zip
# 解压
unzip typecho.zip -d typecho
# 复制到目标路径
sudo cp typecho /var/www/<域名>/

  • 配置nginx: 略。

  • 打开浏览器 https://域名。

  • 问题：目录没有权限

# 简单处理
sudo chmod -R 777 typecho
  • typecho配置文件：config.inc.php
  • 页面出现404， nginx配置有问题
location ~ .*\.php$

改成：

location ~ .*\.php(\/.*)*$

迁移

  • 更换数据库
# config.inc.php文件中修改
  • 更换域名
# 先切换到数据库
use `databases`;
# 修改网站域名，相关的表typecho_options
UPDATE `typecho_options` SET `value` = '新域名地址' WHERE `typecho_options`.`name` = 'siteUrl' AND `typecho_options`.`user` = 0;
# 修改文章地址，相关的表typecho_contents
UPDATE `typecho_contents` SET `text` = REPLACE(`text`,'旧域名地址','新域名地址');
